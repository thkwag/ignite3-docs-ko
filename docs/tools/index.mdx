# 도구

Apache Ignite 3의 명령줄 도구, REST API, 유틸리티를 다룹니다.

## CLI 도구 {#cli-tools}

Ignite CLI는 클러스터 관리, 구성, 운영을 위한 명령어를 제공합니다. 전체 문서는 [CLI 명령어](tools/cli-commands) 참조를 확인하세요.

## REST API {#rest-api}

REST API를 사용하면 HTTP 엔드포인트로 클러스터를 원격 관리하고 모니터링할 수 있습니다. 엔드포인트 문서는 [REST API](tools/rest-api) 참조를 확인하세요.
