# .NET API

C#·F# 애플리케이션용 네이티브 .NET 클라이언트로, 비동기 우선 설계와 LINQ 통합을 지원합니다.

## 개요 {#overview}

.NET API는 Apache Ignite 3 클러스터에 씬 클라이언트로 접근합니다. 모든 연산은 Task와 ValueTask 기반 비동기 패턴을 사용해 논블로킹으로 실행됩니다.

API는 POCO를 사용하는 타입이 지정된 연산과 IIgniteTuple을 사용하는 동적 접근을 모두 지원합니다. LINQ 쿼리는 가능하면 서버 측 실행으로 변환됩니다.
