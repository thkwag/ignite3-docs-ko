# Java API

Apache Ignite 3의 기본 API로, 플랫폼의 모든 기능을 완전하게 지원합니다.

## 개요 {#overview}

Java API는 두 가지 배포 모드를 제공합니다. 씬 클라이언트는 토폴로지에 합류하지 않고 원격 클러스터에 연결합니다. 임베디드 노드는 클러스터 작업과 스토리지에 참여합니다.

두 모드 모두 테이블, SQL, 트랜잭션, 컴퓨트, 스트리밍, 카탈로그 관리를 위한 공통 API를 공유합니다. 임베디드 모드는 클러스터 초기화 API를 추가로 제공하며, 내부 클러스터 컴포넌트에 직접 접근할 수 있습니다.
