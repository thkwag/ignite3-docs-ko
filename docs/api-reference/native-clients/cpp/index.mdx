# C++ API

최신 C++17 기능과 제로카피 연산을 갖춘 고성능 C++ 클라이언트입니다.

## 개요 {#overview}

C++ API는 Apache Ignite 3 클러스터에 씬 클라이언트로 접근합니다. 연산은 비동기 실행에 콜백을 사용하며, 타임아웃 구성을 지원합니다.

API는 템플릿 특수화를 사용하는 타입이 지정된 뷰와, 스키마 없는 연산을 위한 ignite_tuple 기반 바이너리 뷰를 모두 제공합니다. 메모리 관리는 shared_ptr로 자동 리소스 정리를 수행합니다.
