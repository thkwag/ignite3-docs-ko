# 트랜잭션

트랜잭션(transaction)은 `START TRANSACTION` 문으로 시작해 `COMMIT` 문으로 끝나는 일련의 SQL 작업입니다. 모든 작업의 결과가 함께 반영되거나, 아무 결과도 반영되지 않습니다.

:::warning
트랜잭션 제어 문은 [스크립트](/sql/working-with-sql/execute-queries) 안에서만 사용할 수 있습니다.
:::

Apache Ignite 3에서는 `START TRANSACTION` 문으로 트랜잭션을 시작합니다:

<RailroadDiagram>{`
Diagram(
Terminal('START TRANSACTION'),
ZeroOrMore(
Terminal('READ ONLY'),
Terminal('READ WRITE')))
`}</RailroadDiagram>

:::note
트랜잭션 안에서는 DDL 문을 지원하지 않습니다.
:::

매개변수:

- `READ WRITE` - 트랜잭션에서 읽기와 쓰기 작업을 모두 수행할 수 있습니다. 기본값으로 사용됩니다.
- `READ ONLY` - 트랜잭션에서 읽기 작업만 수행할 수 있습니다.

`COMMIT` 문으로 트랜잭션을 닫고 커밋합니다:

<RailroadDiagram>{`
Diagram(
Terminal('COMMIT'))
`}</RailroadDiagram>

## 예시 {#example}

아래 예시는 하나의 트랜잭션에서 테이블에 세 개의 행을 삽입하여, 세 행이 모두 함께 커밋되도록 합니다:

```sql
START TRANSACTION READ WRITE;

INSERT INTO Person (id, name, surname) VALUES (1, 'John', 'Smith');
INSERT INTO Person (id, name, surname) VALUES (2, 'Jane', 'Smith');
INSERT INTO Person (id, name, surname) VALUES (3, 'Adam', 'Mason');

COMMIT;
```
