이 섹션에서는 Apache Ignite 3가 지원하는 모든 데이터 조작어(DML) 명령어를 설명합니다.

## DELETE

테이블에서 데이터를 삭제합니다.

<RailroadDiagram>{`
Diagram(
Terminal('DELETE FROM'),
NonTerminal('qualified_table_name', {href:'./grammar-reference#qualified_table_name'}),
Optional(
Sequence(
Optional('AS'),
NonTerminal('alias'),
)
),
Optional(
Sequence(
NonTerminal('WHERE'),
Terminal('booleanExpression')
)
)
)
`}</RailroadDiagram>

### 매개변수 {#parameters}

- `alias` - 표현식 또는 값에 대한 SQL 별칭입니다.
- `booleanExpression` - 불리언 값을 반환하는 SQL 표현식입니다. `TRUE`가 반환된 레코드만 삭제됩니다. 지정하지 않으면 모든 레코드가 삭제됩니다.

## MERGE

테이블에 데이터를 병합합니다.

<RailroadDiagram>{`
Diagram(
Terminal('MERGE INTO'),
NonTerminal('qualified_table_name', {href:'./grammar-reference#qualified_table_name'}),
Optional(
Sequence(
Optional('AS'),
NonTerminal('alias'),
)
),
Terminal('USING'),
NonTerminal('qualified_table_name', {href:'./grammar-reference#qualified_table_name'}),
Terminal('ON'),
NonTerminal('booleanExpression'),
End({type:'complex'})
)
`}</RailroadDiagram>

<RailroadDiagram>{`
Diagram(
Start({type:'complex'}),
Optional(
Sequence(
NonTerminal('WHEN MATCHED THEN UPDATE SET'),
OneOrMore(Sequence(Terminal('assign', {href:'./grammar-reference#assign'})
),
Terminal(',')
))),
Optional(
Sequence(
NonTerminal('WHEN NOT MATCHED THEN INSERT VALUES'),
NonTerminal('('),
OneOrMore(Sequence(Terminal('value')
),
Terminal(',')
),
NonTerminal(')'),
)))
`}</RailroadDiagram>

:::note
`WHEN MATCHED`와 `WHEN NOT MATCHED` 절 중 적어도 하나는 반드시 있어야 합니다.
:::

### 매개변수 {#parameters-1}

- `alias` - 표현식 또는 값에 대한 SQL 별칭입니다.
- `booleanExpression` - 불리언 값을 반환하는 SQL 표현식입니다. `TRUE`가 반환되면 `WHEN MATCHED` 절이 실행되고, 그렇지 않으면 `WHEN NOT MATCHED` 절이 실행됩니다.
- `value` - 작업 중 테이블에 삽입될 임의의 값입니다.

## UPDATE

테이블의 데이터를 업데이트합니다.

<RailroadDiagram>{`
Diagram(
Terminal('UPDATE'),
NonTerminal('qualified_table_name', {href:'./grammar-reference#qualified_table_name'}),
Terminal('SET'),
OneOrMore(Sequence(Terminal('assign', {href:'./grammar-reference#assign'})
),
Terminal(',')
),
Optional(
Sequence(
Terminal('WHERE'),
NonTerminal('booleanExpression')
)
)
)
`}</RailroadDiagram>

### 매개변수 {#parameters-2}

- `booleanExpression` - 불리언 값을 반환하는 SQL 표현식입니다. `TRUE`가 반환된 레코드만 업데이트됩니다. 지정하지 않으면 모든 레코드가 업데이트됩니다.
