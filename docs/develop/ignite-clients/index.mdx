# Ignite 클라이언트

Apache Ignite 3의 모든 API를 지원하는, 모든 기능을 갖춘 네이티브 클라이언트입니다.

## 기능 {#features}

네이티브 Ignite 클라이언트는 다음 기능을 제공합니다.

- 전체 Table API(RecordView, KeyValueView)
- 준비된 문을 지원하는 SQL API
- 트랜잭션 지원
- 데이터 스트리밍 기능
- 컴퓨트 작업 실행
- 클러스터 토폴로지 정보
- 비동기 작업

## 사용 시점 {#when-to-use}

다음이 필요하다면 네이티브 Ignite 클라이언트를 사용하세요.

- Ignite의 모든 기능에 대한 전체 접근
- 키-값 작업
- 컴퓨트 작업 배포
- 이벤트 처리
- 고성능 데이터 접근

SQL 전용 연결만 필요하다면 [Ignite에 연결](connect-to-ignite/jdbc)을 참조하세요.
